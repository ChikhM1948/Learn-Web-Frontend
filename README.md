## 🎯 À propos de ce dépôt

> **Learn-Web-Frontend** est l'espace pédagogique officiel pour apprendre les bases du **développement web Front-End**.
> Il regroupe **ressources de cours**, **exercices pratiques** et **corrigés**, conçus pour accompagner les étudiants pas à pas, du niveau débutant jusqu'à la réalisation de mini-projets concrets.

Ce dépôt sert de **support de classe vivant** : chaque séance peut s'appuyer sur un dossier dédié, que vous complétiez en cours ou en autonomie chez vous.

<br/>

## 🗂️ Structure du dépôt

```
Learn-Web-Frontend/
│
├── 📁 01-HTML/            → Bases du HTML, structure sémantique
├── 📁 02-CSS/              → Mise en forme, Flexbox, Grid, responsive
├── 📁 03-JavaScript/       → Logique, DOM, événements
├── 📁 exercices/          → Énoncés pratiques par thème
├── 📁 corrections/        → Solutions commentées
├── 📁 ressources/         → Supports, fiches, liens utiles
└── 📁 projets/            → Mini-projets de synthèse
```

## 🧭 Comment l'utiliser

<table>
<tr>
<td width="50%" valign="top">

### 👩‍🎓 Pour les étudiants
1. **Clonez** ou **téléchargez** le dépôt
2. Ouvrez le dossier correspondant à la séance du jour
3. Lisez l'énoncé dans `exercices/`
4. Essayez de résoudre **avant** de consulter `corrections/`
5. Posez vos questions via les *Issues* du dépôt

</td>
<td width="50%" valign="top">

### 👨‍🏫 Pour le formateur
1. Ajoutez les ressources de la séance dans le bon dossier
2. Déposez les exercices avec un fichier `README.md` explicatif
3. Publiez la correction après le délai prévu
4. Utilisez les *Releases* pour figer une version par session

</td>
</tr>
</table>
